# Bad Resume Bullet Anti-Patterns
# These teach the AI what NOT to produce.

- "Worked with Python scripts" — no outcome, no scale, no specificity
- "Helped automate workflows" — passive voice, vague, no ownership
- "Responsible for AI integration" — describes a role, not an achievement
- "Used various tools to improve processes" — which tools? what processes? what improvement?
- "Assisted team with technical solutions" — "assisted" minimizes your contribution
- "Managed data and reporting tasks" — "managed" is vague, no measurable impact
- "Familiar with APIs and automation" — states knowledge, not application
- "Worked on multiple projects simultaneously" — describes work style, not results

# WHY THESE ARE BAD:
# 1. No action verb that shows ownership (Built, Designed, Automated, Reduced)
# 2. No specific tools named (says "tools" instead of "Python + jq + Claude API")
# 3. No quantified outcome (no numbers, no time saved, no scale mentioned)
# 4. No business value connection (doesn't say WHY it mattered)
